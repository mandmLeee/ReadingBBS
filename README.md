# ReadingBBS
讨论与书有关的BBS论坛-书坛

系统由两部分组成：**前台+后台**，前台与普通用户&游客交互；而后台则用于系统管理员进行维护数据库和相关通知操作。
## 前台
1. 登录注销。
2. 用户注册、修改密码。
3. 发布帖子。
4. 评论帖子。
5. 讨论区分类。


## 后台
1. 登录注销。
2. 用户注册、修改密码。
3. 用户、帖子、评论管理。
